# Customer Behavior & Churn Analytics Project

## Project Overview

This project analyzes customer behavior data from a global e-commerce platform to uncover insights related to purchasing patterns, engagement behavior, and churn risk. The analysis focuses on transforming raw data into actionable business intelligence suitable for strategic decision-making.

## Business Objectives
• Understand customer purchasing behavior across demographics
• Identify high-value and high-risk customers
• Analyze return rate and credit balance behavior
• Support churn reduction strategies

## Dataset Description
The dataset contains 50,000 customer records with 25 features covering demographics, engagement metrics, purchasing behavior, financial indicators, and churn status.
- <a href="https://github.com/aabiid07/Customer-Behavior-Churn-Analytics-Project/blob/main/Ecommerce_Customer_Behavior.csv"> Dataset </a>

## Key KPIs
• Total Purchases
• Return Rate
• Credit Balance
• Customer Purchase Rate
• Lifetime Value
• Churn Rate

## Dashboard Insights
Purchase Segmentation shows the USA as the highest revenue contributor. Return Rate analysis highlights higher returns in the USA and UK. Credit Balance analysis indicates strong customer trust in developed markets. Customer lifecycle analysis confirms new customers generate the highest revenue.

## Churn Risk Findings
Customers with low engagement, high inactivity, and high cart abandonment rates show a significantly higher churn probability.

## Business Recommendations
• Focus retention strategies on mid-term customers
• Reduce return rates through better product information
• Implement credit-based loyalty incentives
Conclusion
This project demonstrates how data-driven insights can improve customer retention, increase revenue, and support strategic business decisions.
